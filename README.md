# 西湖论剑数字平台 - 文档说明（前端）

### 技术栈

**vue3** + **vite** + **element-plus** + **scss** + **pinia**

### 文件目录简要

- .husky：格式化/commit提交 钩子
- public：图标、标志性图片
- scripts：pnpm版本要求
- src：
  - api：接口
  - assets：静态资源存放处
  - components：组件
  - routers：路由
  - store：pinia 仓库
  - styles：样式
    - index.scss：全局样式
    - reset.scss：重置样式
    - variable.scss：全局变量
  - utils：封装方法
  - views：各页面 index
  - App.vue：
  - main.ts：入口文件
